微信登录设置
使用说明：

1. APP ID 和 APP KEY 申请方法：需要先申请微信开放平台账号，然后创建网站应用。 详见 微信开放平台
2. 开启方法：在此填写正确的 APP ID 和 APP KEY，点击保存，则微信登录功能添加成功。
3. 关闭方法：APP ID 或 APP KEY 任一值为空，点击保存，则会关闭微信登录（前台看不到）
