消费返利设置
说明：每个用户可以通过自己的邀请链接，邀请好友来网站注册或邀请商家入驻。一旦通过邀请链接注册成功，被邀请用户或商家会成为邀请人的下家。
如果设置相应返利比例，被邀请人的消费或商家的销售行为会给邀请人以返利。返利金额，按返利比率计算。
邀请人未设置与被邀请人的返利比例时，将按照该设置比例值进行返利结算。（注意：无论是购买返利还是卖出返利，前提都是用户去商家验证消费后，才算到我的返利中。只购买没有去消费，暂不记录到我的返利中）
邀请普通用户购买商品，用户消费时，给予邀请人的返利： %
邀请商家入驻平台，商家卖出商品时，给予邀请人的返利： %
返利方式设置： 按产品订单金额计算     按产品利润计算

利润计算公式 : 产品利润 = 订单实付金额 - 商家结算价 = 订单实付金额 - 订单金额 * ( 1-网站分成百分比 )

注册返现设置
说明：每个用户可以通过自己的邀请链接，邀请好友来网站注册。一旦通过邀请链接注册成功，被邀请人会成为邀请人的下家。
如果设置相应返现金额，被邀请人发生首次消费行为时会给邀请人以返现。
返现金额，以下面设置的返现金额计算，留空或设置为0时返现不生效。
邀请新用户注册并消费后返现： 元（邀请一个新用户注册并消费后仅返现一次）
